# Security Policy

## Reporting a Vulnerability

We take the security of BeanPool seriously. If you discover any security vulnerability, please do **NOT** open a public GitHub issue or disclose it publicly. Instead, report it privately to the maintainers.

Please send security reports via email to:
* **security@beanpool.org**

Include as much detail as possible in your report, such as:
- A description of the vulnerability and its potential impact.
- Step-by-step instructions to reproduce the issue.
- Any proof-of-concept code or screenshots.

We will acknowledge receipt of your report within 48 hours and work to resolve the issue promptly in coordination with you.
